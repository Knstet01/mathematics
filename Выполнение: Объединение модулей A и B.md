## Выполнение: Объединение модулей A и B

### Исходное состояние (проблема)

В рефакторинге присутствовало дублирование:
- **Модуль A**: Лемма о чётности квадрата (формулировка в терминах чётности)
- **Модуль B**: Инвариант делимости на 2 (эквивалентная формулировка)

Фактически это одна и та же лемма, что нарушает принцип DRY (Don't Repeat Yourself) в архитектуре доказательства.

---

### Шаг 1: Выбор оптимальной формулировки

Оставляем формулировку в терминах **делимости**, так как она:
1. Более общая (работает не только с чётностью, но и с делимостью на любые числа)
2. Лучше соответствует контексту (в доказательстве используется именно делимость)
3. Легче обобщается (см. границы применимости)

---

### Шаг 2: Объединённый модуль

#### Модуль A: Лемма о делимости квадрата

**Лемма A1 (фундаментальная)**
Если простое число p делит n², то p делит n.

*Доказательство для p = 2:*
Если n нечётно, то n = 2k+1 → n² = 4k²+4k+1 = 2(2k²+2k)+1 → нечётно, следовательно не делится на 2.
Противоречие с условием "2 делит n²". Следовательно, n чётно. ∎

*Примечание:* Для простоты в данном доказательстве достаточно случая p = 2, но лемма сформулирована в общем виде для демонстрации модульности.

**Зависимости:** аксиомы арифметики, определение делимости.

---

### Шаг 3: Модифицированное основное доказательство

#### Модуль B: Основная конструкция

Предположим √2 = p/q в несократимой форме.

Тогда:

1. p² = 2q².
2. Следовательно, 2 делит p².
3. **По лемме A1** (случай p = 2) → 2 делит p → p = 2k.
4. Подстановка: 4k² = 2q² → q² = 2k² → 2 делит q².
5. **По лемме A1** → 2 делит q.
6. Противоречие: p и q делятся на 2, значит дробь сократима.

### Шаг 5: Расширенные границы применимости

Объединённая лемма A1 в формулировке "если простое p делит n², то p делит n" позволяет:

1. **Доказательство иррациональности √3** (аналогичная структура)
2. **Доказательство иррациональности √p** для любого простого p
3. **Обобщение на любые числа, не являющиеся полными квадратами** (через основную теорему арифметики)

---

### Шаг 6: Итоговая архитектура

```
┌─────────────────────────────┐
│     Модуль A (Лемма)        │
│  Если p|n² (p простое),     │
│  то p|n                     │
│  (доказано для p=2)         │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│     Модуль B (Основное)     │
│  √2 = p/q (несократима)     │
│  → p² = 2q²                 │
│  → 2|p² → (по A) → 2|p      │
│  → p=2k → 2k² = q²          │
│  → 2|q² → (по A) → 2|q      │
│  → противоречие             │
└─────────────────────────────┘
```

**Зависимости:**
- Модуль B зависит только от Модуля A
- Модуль A зависит только от базовой арифметики
- Циклических зависимостей нет
